Customer Shopping Behavior Analysis
Python · Pandas · SQL · PostgreSQL · Power BI · Seaborn · Matplotlib

End-to-end analytics project analyzing 3,900 transactions across 18 features to uncover customer purchasing patterns, segment behavior, and drive business recommendations.

Key Findings:

💰 Male customers generated 2× more revenue ($157,890 vs $75,191)
👥 3,116 Loyal customers identified — largest and most valuable segment
📉 Subscription status doesn't drive higher per-order spend ($59.87 vs $59.49)
📦 73% non-subscribers contributed ~$170K vs 27% subscribers at $62.6K
What I did:

Cleaned and preprocessed data using Pandas (missing value imputation, snake_case normalization)
Engineered age_group and purchase_frequency_days features for segmentation
Exported cleaned DataFrame to PostgreSQL for SQL-based analysis
Built interactive Power BI dashboard with subscription, gender, category & shipping filters
